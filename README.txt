Проект Broker
---------------

В проекте производится расчет
комиссионного вознаграждения брокера,
в зависимости от объема продаж, по
прогрессивной шкале.

Данный проект на развитие и
закрепление навыков по:
- переменные,
- методы и переопределение методов,
- конструкторы,
- наследование,
- интерфейсы.

1) ВНИМАТЕЛЬНО прочитайте комментарии в коде.
2) Завершите проект по шаблону.
3) Архив завершенного проекта загрузите в LMS.
